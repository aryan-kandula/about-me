# Portfolio — Aryan Kandula

**Live site:** [aryan-kandula.github.io/portfolio](https://aryan-kandula.github.io/portfolio/)

---

A personal portfolio website I built from scratch using HTML, CSS, and vanilla JavaScript. Designed to showcase my projects, skills, and background as a Computer Science student and developer.

## What's in it

- **Animated hero section** with a custom teal/dark theme and floating background orbs
- **Skills section** with animated progress bars that trigger on scroll
- **Projects section** covering my main work — web apps, Java desktop software, SQL projects, Linux systems work, and GitHub collaboration
- **Education section** showing my current A.S. at FTCC and planned transfer to UNCP
- **Fixed navigation bar** that highlights the active section as you scroll and smoothly jumps to each section on click
- **Custom cursor** with a trailing ring effect
- **3D tilt effect** on project cards on hover
- **Fully responsive** layout for mobile and desktop

## Built with

- HTML5
- CSS3 — custom properties, glassmorphism, keyframe animations
- Vanilla JavaScript — IntersectionObserver, scroll events, custom cursor, ripple effects
- Google Fonts — DM Sans + DM Mono

## Run locally

Just open `index.html` (or the `.html` file) in any browser. No build step, no dependencies.
